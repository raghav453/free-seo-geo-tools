# Contributing

Thanks for helping improve this resource.

This repository is meant to stay simple, useful, and non-promotional.

## What You Can Suggest

You can open an issue for:

- Broken tool links
- Incorrect descriptions
- Better category placement
- Suggestions for new free SEO, GEO, schema, hreflang, or localization tools
- Improvements to README clarity

## Content Guidelines

Please keep descriptions:

- Clear
- Short
- Useful
- Non-promotional
- Relevant to SEO, technical SEO, multilingual SEO, structured data, localization, or AI search visibility

## Example Format

```markdown
- [Tool Name](Tool URL) - One short sentence explaining what the tool helps users do.
